// Architecture

How It Works

Five stages transform a raw camera frame into readable text in under 50 milliseconds.

📷
01
Webcam Capture
Browser grabs 720p frames via getUserMedia API.
✋
02
Landmark Detection
MediaPipe WASM extracts 21 3D landmarks per frame.
📐
03
Feature Engineering
Finger states and joint angles computed in JS.
🧠
04
Classification
Pattern lookup maps features to ASL/ISL characters.
💬
05
Text Output
7-frame stabiliser confirms gestures into words.

// Capabilities

What It Can Do

🇮🇳
Indian Sign Language
Full Devanagari character support — vowels, consonants, and digits with context-aware disambiguation.
🇺🇸
American Sign Language
All 26 English letters. Pattern logic separates look-alike handshapes like C/S and O/fist.
⚡
Real-Time, In-Browser
MediaPipe WASM runs entirely client-side. No server, no Python, no install required.
📊
Confidence Scoring
Every prediction scored across landmark spread, hand steadiness, and pattern exclusivity.
✍️
Word & Sentence Builder
Hold-to-confirm prevents accidental input. Letters build words; words build sentences.
🖐️
Finger State Display
Live visual showing which fingers the model detects as extended — see what the model sees.
// Stack

Technology Used

MediaPipe Hands
21-point 3D landmark detection. Runs as WASM — same model as the Python notebook version.
OpenCV / Python
Full-accuracy backend. Adds curvature angles, stabilisation and TTS output.
NumPy
Vector math for joint angles, curvature and steadiness in the Python pipeline.
Vanilla JS
No framework overhead. Direct canvas drawing, frame buffer, keyboard controls.
